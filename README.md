# Français 🇫🇷
<details open>
  <summary><b>Qu'est-ce que c'est ?</b></summary>
  <p>Des listes d'IP et de réseaux IP d'opérateurs ou hébergeurs ou bots ou services Web.</p>
</details>
<details>
  <summary><b>Quelle est la liste de fichiers ?</b></summary>
  <p>

### <b>IP lists</b> :
Différentes listes d'adresses IP utilisées par les services Web :

- Censys (scanner)
- DuckDuckGo (web search engine bot)
- Let's Encrypt (IP used to validate web servers)
- Ngrok
- OVH Shared Web Servers
- UptimeRobot

### <b>Subnets lists</b> :
Différentes listes de réseaux IP de l'AS d'opérateurs ou d'hébergeurs ou de robots :

- Orange AS3215
- SFR AS15557
- Bouygues Telecom AS5410
- Free AS12322
- OVH AS16276
- Google Bot
- Bing Bot
- CloudFlare (CDN only)
- AWS CloudFront (CDN only)
- Faslty (CDN)
- Akamai (CDN only)
  </p>
</details>

<details open>
  <summary><b>What is this ?</b></summary>

  <summary><b>What is the file list?</b></summary>

### <b>IP lists</b> :

- DuckDuckGo (web search engine bot)
- Let's Encrypt (IP used to validate web servers)
- Ngrok
- OVH Shared Web Servers
- UptimeRobot
